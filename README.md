# ddh
Deletes duplicates in .bash_history

## Installation

```
go get github.com/usk81/ddh
```

## Usage

```bash
# deletes duplicates in .bash_history
$ ddh

# show version
$ ddh verison

# help
$ ddh help
delete duplicates in .bash_history

Usage:
  ddh [flags]
  ddh [command]

Available Commands:
  help        Help about any command
  version     show version

Flags:
  -h, --help   help for ddh

Use "ddh [command] --help" for more information about a command.
```
